Circular Dependencies

The require function given earlier in this chapter supports this type of dependency cycle.

Can you see how it handles cycles?

- Since "require" adds modules before it starts loading whatever call is made, the current interface would be returned.

What would go wrong when a module in a cycle does replace its default exports object?

- If a module in a cycle replace its default exports object, another module receiving its interface value would receive the default instead of the intented value.
